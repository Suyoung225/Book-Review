
현재 사용 중인 셸의 작업 디렉토리 위치 출력 <br>
$ pwd

파일 존재 여부 확인 <br>
$ ls file

파일 줄 번호와 함께 출력 <br>
$ cat **-n** file

파일 화면 단위로 보기 <br>
$ more file

파일 일부 보기 <br>
head, tail

$ head -10 file

추가되는 내용 실시간으로 보기 <br>
$ tail -f file

파일 복사 <br>
$ cp sourcefile targetfile

$ cp source1 source1 ... directory

cp는 이름 같으면 덮어쓰기, 이를 방지하려면 -i 추가 (overwrite 여부 물어봄)

디렉토리 복사 <br>
디렉토리 자체 복사는 안돼서 재귀적 복사만 가능 <br>
$ cp -r sourceDir targetDir

하위 디렉토리 확인  <br>
$ ls -R dir
