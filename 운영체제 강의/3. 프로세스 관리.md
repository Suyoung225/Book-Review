# 커널 주소 공간
## code
- 시스템 콜, 인터럽트 처리 코드
- 자원 관리를 위한 코드
- 편리한 서비스 제공을 위한 코드

## data
PCB(Process Control Block): 모든 하드웨어, 프로세스를 관리하기 위한 자료구조를 가짐

## stack
커널 함수를 호출시킨 프로세스별로 별도의 스택을 둠 ex) Process A의 커널 스택, Process B의 커널 스택

**사용자 정의 함수**

**라이브러리 함수**

**커널 함수**
